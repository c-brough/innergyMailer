Azure App Registration — one-time setup
  1. Go to portal.azure.com → App registrations → New registration.
    Name: anything (e.g. Innergy Mailer)
    Supported account types: Accounts in any organizational directory and personal Microsoft accounts
    Redirect URI: leave blank → click Register
  2. Copy the Application (client) ID from the overview page.
  3. Under API permissions → Add a permission → Microsoft Graph → Delegated → add Mail.ReadWrite.
  4. Paste the Client ID below and click Sign in. A browser tab opens — sign in with your Microsoft account. The token is cached so you only do this once.

New Outlook requires a one-time Microsoft sign-in via the Graph API. Register a free Azure app and paste the Application (client) ID below.
